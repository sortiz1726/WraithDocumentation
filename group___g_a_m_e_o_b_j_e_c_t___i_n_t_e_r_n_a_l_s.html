Detailed Description

Game Object is the central entitiy of Engine for deriving and creating user define object for the game.

Game Object derives from Updatable, Drawable, Alarmable, Inputable, and Collidable. The main purpose of this object is to provide centralize hub for all functions from derived class and registration/deregistration for the Game Object itself.
